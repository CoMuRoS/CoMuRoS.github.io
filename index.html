Quick Start Guide

Complete installation and setup instructions for development and production environments

üìã System Requirements

Operating System: Ubuntu 20.04 LTS / Ubuntu 22.04 LTS / Docker (Recommended)

Hardware: 16GB RAM (minimum 8GB), 8-core CPU, NVIDIA GPU (optional but recommended)

Software: ROS Noetic/Humble, Python 3.8+, CMake 3.16+, Git 2.25+

üöÄ One-Line Installation

# Quick install script (recommended for new users)
curl -fsSL https://raw.githubusercontent.com/yourusername/comuros/main/install.sh | bash

# Alternative: Docker deployment
docker run -it --privileged --net=host comuros/comuros:latest

üõ†Ô∏è Manual Installation

# Install ROS Noetic (Ubuntu 20.04)
sudo sh -c 'echo "deb http://packages.ros.org/ros/ubuntu $(lsb_release -sc) main" > /etc/apt/sources.list.d/ros-latest.list'
curl -s https://raw.githubusercontent.com/ros/rosdistro/master/ros.asc | sudo apt-key add -
sudo apt update && sudo apt install ros-noetic-desktop-full

# Setup ROS environment
sudo rosdep init && rosdep update
echo "source /opt/ros/noetic/setup.bash" >> ~/.bashrc
source ~/.bashrc

üì• Clone and Build

# Create and setup workspace
mkdir -p ~/comuros_ws/src && cd ~/comuros_ws/src

# Clone repositories
git clone --recursive https://github.com/yourusername/comuros.git
git clone https://github.com/yourusername/comuros_msgs.git

# Install dependencies and build
cd ~/comuros_ws
rosdep install --from-paths src --ignore-src -r -y
catkin_make -j$(nproc) || catkin build

# Source workspace
echo "source ~/comuros_ws/devel/setup.bash" >> ~/.bashrc
source ~/.bashrc

Platform Customization

Extend CoMuRoS with custom robot configurations and AI models

ü§ñ Custom Robot Configuration

Integrate your custom robot platforms with CoMuRoS coordination system

# Step 1: Create robot configuration file
mkdir -p ~/comuros_ws/src/comuros/config/robots/
cp comuros/config/robots/template_robot.yaml my_custom_robot.yaml

# Step 2: Edit configuration parameters
nano ~/comuros_ws/src/comuros/config/robots/my_custom_robot.yaml

‚öôÔ∏è Robot Configuration Template

# my_custom_robot.yaml
robot_name: "custom_robot_001"
robot_type: "mobile_manipulator"

# Physical properties
dimensions:
  length: 1.2    # meters
  width: 0.8     # meters
  height: 1.5    # meters
  
# Motion capabilities
max_linear_velocity: 2.0    # m/s
max_angular_velocity: 1.5   # rad/s
acceleration_limits:
  linear: 1.0    # m/s¬≤
  angular: 2.0   # rad/s¬≤

# Sensor configuration
sensors:
  - type: "lidar"
    topic: "/scan"
    range: 30.0
  - type: "camera"
    topic: "/camera/image_raw"
    
# Task capabilities
capabilities:
  - "navigation"
  - "manipulation"
  - "object_detection"
  
# Communication settings
communication:
  protocol: "ros"
  namespace: "/custom_robot_001"

üöÄ Register and Launch Custom Robot

# Step 3: Register robot with CoMuRoS
rosrun comuros register_robot.py --config my_custom_robot.yaml

# Step 4: Launch robot interface
roslaunch comuros custom_robot_interface.launch robot_config:=my_custom_robot

# Step 5: Verify registration
rosservice call /comuros/list_robots

# Step 6: Include in coordination missions
roslaunch comuros multi_robot_mission.launch \
  robots:="turtlebot3_1,turtlebot3_2,custom_robot_001"

üß† Custom LLM Integration

Integrate custom Large Language Models for natural language task planning

# Step 1: Create LLM configuration
mkdir -p ~/comuros_ws/src/comuros/config/llm/
nano ~/comuros_ws/src/comuros/config/llm/custom_llm_config.yaml

üîß LLM Configuration Template

# custom_llm_config.yaml
llm_name: "custom_gpt_model"
model_type: "openai_compatible"

# Model connection settings
api_settings:
  endpoint: "https://your-custom-endpoint.com/v1"
  api_key: "${CUSTOM_LLM_API_KEY}"
  model_name: "your-custom-model-v1"
  
# Model parameters
parameters:
  temperature: 0.3
  max_tokens: 2048
  top_p: 0.9
  
# CoMuRoS integration
task_planning:
  enabled: true
  system_prompt: |
    You are a multi-robot coordination assistant.
    Parse natural language commands into structured robot tasks.
    
response_format:
  type: "structured"
  schema: "comuros_task_schema_v1"
  
# Fallback settings
fallback:
  model: "default_comuros_planner"
  timeout: 30.0

üéØ Deploy Custom LLM

# Step 2: Set API key environment variable
export CUSTOM_LLM_API_KEY="your-api-key-here"
echo 'export CUSTOM_LLM_API_KEY="your-api-key-here"' >> ~/.bashrc

# Step 3: Register LLM with CoMuRoS
rosrun comuros register_llm.py --config custom_llm_config.yaml

# Step 4: Launch LLM service
roslaunch comuros llm_service.launch llm_config:=custom_llm_config

# Step 5: Test natural language commands
rosservice call /comuros/natural_language_task \
  "Move robot 1 to the warehouse and pick up the blue box"

# Step 6: Monitor LLM performance
rostopic echo /comuros/llm_metrics

# Test LLM integration
